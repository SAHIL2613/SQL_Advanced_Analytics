# SQL Advanced Analytics Assignment

## Objective
To perform advanced analytics on the Superstore dataset using SQL by implementing Subqueries, CTEs, and Window Functions to derive meaningful business insights and improve query structuring efficiency.


## Tools Used
- MySQL Workbench
- Superstore Dataset


## Tasks Completed
- Imported dataset into `superstore_raw`
- Created `customers`, `orders`, and `products` tables
- Used Subqueries
- Used CTEs
- Used Window Functions (`ROW_NUMBER`, `RANK`, `DENSE_RANK`)
- Completed Customer Sales Insights 


## Key Insights
- A small number of customers contribute a significant portion of total sales.

- Several customers placed only one order, indicating opportunities for customer retention.

- Top-ranked customers generated substantially higher sales than the average customer.

- Window Functions made customer ranking and order analysis efficient.

- CTEs improved the readability and organization of aggregation queries.
